# Power-BI-dashboard

Power BI Sales Dashboard (Superstore Dataset)
. Designed an interactive Power BI dashboard to analyze sales, profit, segment-wise revenue, and market insights.

. Performed data cleaning & transformation using Power Query and created DAX measures for KPIs (Sales, Profit, Quantity, Discount).

. Identified key business trends such as top-performing categories, high-profit regions, and yearly sales growth.

. Built multiple visualizations (bar charts, donut chart, KPIs, line chart) to enable data-driven decision-making.

. Published dashboard and documented findings for clear business storytelling.
